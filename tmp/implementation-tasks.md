# ベトナム語学習アプリ - 実装タスクリスト

このドキュメントは、[PRD](./vietnamese-learning-app-prd.md) を基に作成された実装タスクリストです。
Phase 1 (MVP) を中心に、実装可能な粒度まで分解しています。

## タスク凡例

- **優先度**
  - `P0`: クリティカル（MVP必須）
  - `P1`: 高（MVP推奨）
  - `P2`: 中（Phase 2）
  - `P3`: 低（Phase 3以降）

- **サイズ見積もり**
  - `S`: 小（1-2時間）
  - `M`: 中（半日-1日）
  - `L`: 大（2-3日）
  - `XL`: 特大（1週間以上）

## リスク管理

### 高リスクタスク

1. **TASK-107: 単語音声ファイルの準備**
   - **リスク**: ネイティブスピーカーの確保が困難、音質が低い
   - **対策**:
     - 早期にネイティブスピーカーを確保（ベトナム人同僚に依頼）
     - 代替として高品質TTS（Google Cloud TTS）を利用
     - 音声録音のガイドラインを作成

2. **TASK-402: フラッシュカードコンポーネント**
   - **リスク**: スワイプジェスチャーの実装が複雑、パフォーマンス問題
   - **対策**:
     - 既存ライブラリの活用（react-swipeable, framer-motion）
     - 早期プロトタイプで技術検証
     - パフォーマンステストを並行実施

3. **TASK-601: 学習統計ページ**
   - **リスク**: グラフ描画のパフォーマンス、データ集計ロジックの複雑さ
   - **対策**:
     - 軽量グラフライブラリの選定（Recharts, Chart.js）
     - データ集計ロジックを事前設計
     - 大量データでのパフォーマンステスト

### 外部依存リスク

- **音声ファイル**: ネイティブスピーカーの協力が不可欠
- **デザイン素材**: アイコン、イラストの調達
- **コンテンツ検証**: すべてのベトナム語コンテンツの正確性チェック
