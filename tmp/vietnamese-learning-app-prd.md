# ベトナム語学習アプリ - PRD（プロダクト要求仕様書）

## 1. プロダクト概要

### 1.1 目的
ベトナム語の文字の読み方もわからない初心者が、楽しく基礎を学習できるアプリケーションを開発する。自分自身や会社の同僚が使用し、ベトナム語でのコミュニケーション能力を向上させることを目指す。

### 1.2 ビジョン
誰でも気軽にベトナム語の基礎を習得でき、実践的なコミュニケーションができるようになるための学習プラットフォームを提供する。

## 2. ターゲットユーザー

### 2.1 プライマリーユーザー
- ベトナム語の学習経験がほとんどない日本人ビジネスパーソン
- ベトナム語の文字（声調記号付きラテン文字）の読み方がわからない初心者
- 会社でベトナム人の同僚とコミュニケーションを取る必要がある人

### 2.2 ユーザーペルソナ
**田中太郎（30歳、会社員）**
- ベトナム支社との業務が増えたが、ベトナム語は全く話せない
- スキマ時間に効率的に学習したい
- 文法より実践的な会話を優先したい
- ゲーム感覚で楽しく続けられる学習方法を求めている

## 3. 主要機能

### 3.1 文字・発音学習機能
**優先度：高**

#### 3.1.1 ベトナム語アルファベット学習
- 29文字のベトナム語アルファベットの紹介
- 各文字の発音を音声で再生
- 声調記号（6種類）の説明と聞き分け練習
- インタラクティブな発音練習

#### 3.1.2 声調トレーニング
- 6つの声調（平声、鋭声、玄声、問声、跌声、重声）の視覚的表現
- 音声を聞いて声調を当てるクイズ
- 同じ単語で声調が異なる例の比較

### 3.2 単語カード機能
**優先度：高**

#### 3.2.1 フラッシュカード
- カテゴリー別の単語カード（挨拶、数字、色、食べ物、ビジネス用語など）
- 表：ベトナム語、裏：日本語訳＋読み方（カタカナ）
- 音声再生機能
- スワイプ操作で「覚えた」「まだ」を仕分け

#### 3.2.2 単語帳管理
- よく使う単語リスト（頻出Top100など）
- 業種別単語集（IT、製造、サービス業など）
- 自分で単語を追加できるカスタムデッキ機能
- 復習が必要な単語の自動提示（間隔反復学習）

### 3.3 フレーズ学習機能
**優先度：中**

#### 3.3.1 シチュエーション別フレーズ
- 日常会話（挨拶、自己紹介、お礼など）
- ビジネス会話（会議、電話、メールなど）
- レストラン・買い物
- 緊急時のフレーズ

#### 3.3.2 ロールプレイ練習
- シチュエーションを選んで会話練習
- 音声認識による発音チェック（将来機能）

### 3.4 ゲーミフィケーション機能
**優先度：高**

#### 3.4.1 進捗管理
- 学習日数のストリーク表示
- レベルシステム（初級→中級→上級）
- 経験値とレベルアップ
- 達成バッジ（「7日連続学習」「100単語マスター」など）

#### 3.4.2 デイリーチャレンジ
- 毎日の学習目標設定（5分、10分、15分など）
- 日替わりクイズ
- 連続学習ボーナス

#### 3.4.3 クイズモード
- 4択クイズ（日本語→ベトナム語、ベトナム語→日本語）
- リスニングクイズ（音声を聞いて意味を選ぶ）
- スペリングクイズ（音声を聞いて綴りを入力）
- タイムアタックモード

### 3.5 学習統計・分析機能
**優先度：中**

- 学習時間の記録とグラフ表示
- 習得済み単語数の表示
- 弱点分野の分析と推奨学習
- 週次・月次レポート

### 3.6 チーム学習機能
**優先度：低（将来機能）**

- 社内メンバー間でのランキング表示
- チームでの学習目標設定
- メンバー間で単語集の共有

## 4. 技術要件

### 4.1 プラットフォーム
- Webアプリケーション（レスポンシブデザイン）
- デスクトップ、タブレット、スマートフォン対応
- 将来的にPWA化を検討（オフライン対応）

### 4.2 技術スタック（推奨）
- **フロントエンド**: React / Next.js
- **バックエンド**: Node.js / Express（必要に応じて）
- **データベース**: Firebase / PostgreSQL
- **音声再生**: Web Audio API / Howler.js
- **状態管理**: Redux / Context API / Zustand

### 4.3 外部API・サービス
- 音声合成：Google Text-to-Speech API または Web Speech API
- 音声認識（将来機能）：Google Speech-to-Text API
- 認証：Firebase Authentication

### 4.4 パフォーマンス要件
- ページ読み込み時間：3秒以内
- 音声再生の遅延：500ms以内
- オフライン時の基本機能利用（PWA化後）

## 5. UI/UX要件

### 5.1 デザイン原則
- **シンプル**: 初心者でも迷わない直感的なUI
- **明るく楽しい**: 学習意欲を高めるカラフルな配色
- **モバイルファースト**: スマホでの使いやすさを最優先
- **アクセシビリティ**: 文字サイズ調整、高コントラストモード

### 5.2 画面構成

#### 5.2.1 ホーム画面
- 今日の学習目標と進捗バー
- クイックアクセス（単語カード、クイズ、フレーズ練習）
- 連続学習日数とレベル表示
- デイリーチャレンジ

#### 5.2.2 単語カード画面
- カテゴリー選択
- カードのめくり/スワイプ操作
- 音声再生ボタン
- 進捗表示（残り枚数）

#### 5.2.3 学習画面
- ナビゲーションメニュー（文字、単語、フレーズ、クイズ）
- 各学習コンテンツ
- 戻る/次へボタン

#### 5.2.4 統計画面
- グラフとチャート
- 習得単語リスト
- 学習履歴

### 5.3 カラースキーム
- プライマリーカラー：ベトナムの国旗にちなんだ赤系
- セカンダリーカラー：明るい黄色、緑
- バックグラウンド：白または明るいグレー
- アクセントカラー：青（リンク、ボタン）

## 7. 学習コンテンツ構成

### 7.1 レベル1：超初級（文字と発音）
1. ベトナム語アルファベット29文字
2. 声調記号の基礎
3. 基本的な挨拶10フレーズ
4. 数字1-20
5. よく使う名詞30語

### 7.2 レベル2：初級（日常会話基礎）
1. 自己紹介フレーズ
2. 日常会話頻出単語100語
3. 基本的な動詞50語
4. 色、曜日、月などの基本語彙

### 7.3 レベル3：初中級（ビジネス基礎）
1. ビジネス挨拶とマナー

3. 会議でよく使う表現
4. メールでの定型表現
5. 業種別専門用語

## 8. 成功指標（KPI）

### 8.1 ユーザーエンゲージメント
- デイリーアクティブユーザー（DAU）

### 8.2 学習効果
- 100単語習得までの平均日数：30日以内
- クイズ正解率の向上：初回60%→1ヶ月後80%
- ユーザー自己評価：「ベトナム語が少し話せるようになった」70%以上

### 8.3 ユーザー満足度
- アプリストア評価：4.0以上
- ユーザーレビューでの肯定的フィードバック：80%以上
- 友人・同僚への推奨意向（NPS）：40以上

## 10. リスクと対策

### 10.1 技術的リスク
| リスク | 影響 | 対策 |
|--------|------|------|
| 音声データのサイズが大きくなる | 読み込みが遅い | 音声ファイルの圧縮、CDN利用、遅延読み込み |
| ベトナム語の声調を正確に音声合成できない | 学習効果が下がる | ネイティブスピーカーの音声録音を使用 |
| オフライン機能の実装が複雑 | 開発期間延長 | Phase 3に延期、まずはオンライン版に注力 |

### 10.2 コンテンツリスク
| リスク | 影響 | 対策 |
|--------|------|------|
| 翻訳や発音表記に誤りがある | ユーザーが間違った知識を習得 | ネイティブチェックを必須化 |
| ビジネス用語が実務で使われない | 実用性が低い | ベトナム人同僚へのヒアリング |

### 10.3 ユーザーリスク
| リスク | 影響 | 対策 |
|--------|------|------|
| ユーザーが途中で飽きる | 継続率低下 | ゲーミフィケーション要素の強化 |
| 初心者に難しすぎる | 離脱率増加 | 超初級コンテンツの充実、段階的な難易度設定 |

## 12. 成功のための重要ポイント

1. **初心者への徹底的な配慮**
   - 文字の読み方から丁寧に説明
   - 挫折しない難易度設定
   - カタカナ読み仮名を必ず表示

2. **楽しさの追求**
   - ゲーム要素の充実
   - 視覚的に楽しいデザイン
   - 達成感を感じられる仕組み

3. **実用性の重視**
   - 実際のビジネスシーンで使える表現
   - 社内のベトナム人同僚との協力
   - 頻出語彙を優先

4. **継続しやすい設計**
   - 1日5-10分から始められる
   - スキマ時間に学習可能
   - モバイルでの使いやすさ

5. **品質の担保**
   - ネイティブチェック必須
   - 音声は可能な限り人間の録音を使用
   - 定期的なコンテンツの見直しと改善

## 付録A: 参考情報

### ベトナム語の基礎知識
- **文字**: ラテン文字ベース（29文字）
- **声調**: 6種類（平声、鋭声、玄声、問声、跌声、重声）
- **語順**: SVO（主語-動詞-目的語）で日本語と異なる
- **特徴**: 孤立語で単語が変化しない（活用・語尾変化なし）

### 競合分析
- Duolingo: ゲーミフィケーションが優れているが日本語話者向けベトナム語コースが少ない
- Memrise: 単語学習に特化しているが体系的な文法学習が弱い
- Ling App: ベトナム語対応だが日本語UIがない

### 差別化ポイント
- 日本人ビジネスパーソンに特化
- ビジネスシーンでの実用性重視
- 社内学習グループでの利用を想定
- カタカナ読み仮名で初心者にやさしい
